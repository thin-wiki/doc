
## 请求URL
- `http://localhost:9887/api/admin/storage/gitee/{storageId}`

## 请求方式
- DELETE

## 请求参数

|参数名|类型|说明|
|:-----  |:-----|-----|
|storageId |string   | 记录id，请求URL中的{storageId}  |

## 返回示例

``` json
{
    "code": 0, 
    "msg": "success"
}
```

## 返回参数说明

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |返回码，0-正常，1-异常  |
|msg |string   | 返回消息  |
